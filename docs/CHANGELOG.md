# 0.3.0

Релиз от 25.09.2025. Изменения собраны по diff между тегами `v0.1.9` и `v0.3.0`.

- Вокруг Markdown- и GitHub-сценариев появился отдельный сервисный слой: в проект вошли `bot/service.py` и `bot/github_service.py`, поэтому рендер и просмотр материалов стали менее завязаны на один большой обработчик.
- К существующему LaTeX-рендеру добавилась полноценная поддержка Mermaid-схем и подготовка Puppeteer-конфигурации для headless-рендера.
- Заметно расширился сценарий чтения GitHub-конспектов: стали взрослее режимы вывода Markdown, улучшилась работа поиска по репозиториям и настройка пользовательских GitHub-источников.
- Обновлены Docker-артефакты, README и визуальные материалы проекта; в этот диапазон также попали промежуточные изменения ветки `0.2.x`, которые не были оформлены отдельными тегами.

# 0.3.3
